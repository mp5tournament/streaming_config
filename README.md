# 直播配置生成器

直播配置生成器，使用本项目的[发布版](https://mp5tournament.github.io/streaming_config/)，可以根据填入的锦标赛信息生成相应的直播配置bracket.json，另外也可以一键获取最新MP5直播配置。  
不包含队旗图片本体，生成的配置中队旗命名为A、B、C……等。  
目前不提供生成赛程功能，谁写一个pr吧（
